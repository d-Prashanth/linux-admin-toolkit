# Linux Admin Toolkit

A modular Bash CLI tool for Linux user management.

## Features

- Create users
- Delete users
- List users
- Random password generation
- Forced password change
- Logging
- Modular Bash structure

## Commands

userctl add user1,user2,user3
userctl delete user1,user2
userctl list
